# 安装及使用


## 安装

cec-cli 项目基于 node，请确保已具备较新的 node 环境（>=8.0.0），推荐使用 node 版本管理工具 nvm 来管理 node，
这样不仅可以很方便地切换 node 版本，而且全局安装时候也不用加 sudo 了。



### CLI 工具安装

首先，你需要使用 npm 或者 yarn 全局安装`@cecdataFE/cec-cli`，或者直接使用npx:

```bash
  使用 npm 安装 CLI
  $ npm install -g @cecdataFE/cec-cli
  OR 使用 yarn 安装 CLI
  $ yarn global add @cecdataFE/cec-cli
  OR 安装了 cnpm，使用 cnpm 安装 CLI
  cnpm install -g @cecdataFE/cec-cli
```

## 常用 CLI 命令

### 查看 cec-cli 版本号

```bash
 $ cec-cli --version
```

### 查看 cec-cli 所有命令及帮助

```bash
 $ cec-cli --help
```
